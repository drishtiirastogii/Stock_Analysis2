# Setup Instructions

1. Create a virtual environment:
   python -m venv venv

2. Activate the virtual environment:
   # Windows
   .\venv\Scripts\activate

3. Install dependencies:
   pip install -r requirements.txt

4. Run the app:
   streamlit run app_yahoo_UI.py
